# leetcode-helper MCP Server

An MCP server that provides LeetCode DSA guidance by calling Claude under the hood.

## Tools

### `get_approach(problem)`
Returns the recommended DSA approach for a given problem.
- **Input:** Full problem statement
- **Output:** Pattern name, algorithm steps, time/space complexity, and why this approach fits

### `get_hint(problem, stuck_on)`
Returns a targeted hint without revealing the solution.
- **Input:** Problem statement + description of where the user is stuck
- **Output:** Key missing insight, a Socratic guiding question, a small concrete hint, and what to try next

### `get_complexity(problem, solution_code)`
Analyzes time and space complexity of the user's code.
- **Input:** Problem statement + user's solution code
- **Output:** Per-section complexity breakdown, overall Big-O, and whether optimization is possible

## Setup

```bash
npm install
npm run build
```

## Running

```bash
npm start
# or directly:
node dist/index.js
```

## Adding to Claude Desktop / Claude Code

In your MCP config (e.g. `claude_desktop_config.json` or `.claude/settings.json`):

```json
{
  "mcpServers": {
    "leetcode-helper": {
      "command": "node",
      "args": ["d:/leetcode-mcp/dist/index.js"]
    }
  }
}
```

## Environment

- Requires `ANTHROPIC_API_KEY` in environment
- Uses `claude-sonnet-4-6` model for all three tools
- Transport: stdio (standard MCP)
